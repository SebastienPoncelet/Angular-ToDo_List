# Launch the app
Make sure to enter the project folder with:
`cd ./my-todo-list`
Install all NPM packages:
`npm install`
Enter the following command line in the terminal:
`ng serve`
Go to this URL in your browser:
`http://localhost:4200/`

Have fun!

# Tests
Enter the following command line in the terminal:
`npm run test`
